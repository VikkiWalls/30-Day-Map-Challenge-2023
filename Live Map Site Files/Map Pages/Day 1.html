30 Day Map Challenge 2022

Day 27 - Music

Github Instagram Mastodon X

The goal of this map was to plot the locations that Dolly Parton has performed in over the years. Venue names were sourced from the concert archives site and OpenStreetMap was used to find the latitude and longatude value for each venue.

During the 30 day map challenge there was a significant time constraint and given the co-ordinates for each concert venue were sourced and compiled by hand, the map produced only showed venues between 1972 and 2007. Since then the remaining data has been compiled, the code re run and there is now a map up to date to 2022. Both versions of the map can be accessed below:
